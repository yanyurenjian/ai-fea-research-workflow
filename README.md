# AI-有限元分析研究工作流程

## 项目简介

本项目用于记录 AI 辅助有限元科研工作的流程。研究方向主要是涂层/基体体系在熔覆后冷却阶段，由于热膨胀系数不匹配产生热应力，从而引发涂层开裂和界面剥离的问题。

本项目并不是一个完整商业软件，而是一个 AI 辅助科研建模、脚本编写、后处理分析和论文写作的工作流示例。

## 研究背景

在涂层/基体结构中，熔覆后冷却阶段会产生明显的热失配应力。该应力可能导致两类典型失效：

- 涂层或基体内部开裂
- 涂层与基体界面剥离

为了分析不同失效路径之间的竞争关系，本研究使用有限元方法建立二维局部模型，并结合 cohesive zone model 对界面和层内损伤进行分析。

## AI 辅助内容

AI 主要参与以下工作：

- 文献阅读与论文内容总结
- 有限元建模思路梳理
- Abaqus Python 脚本生成与修改
- 多组参数模型的批量计算设计
- 应力场、位移场和损伤变量后处理
- 解析解与有限元结果对比
- 论文小节、表格和图题整理

## 当前工作流

1. 建立涂层/基体二维有限元模型。
2. 设置材料参数、热膨胀系数和界面 cohesive 参数。
3. 通过 Abaqus Python 脚本批量修改参数。
4. 提交多组有限元计算任务。
5. 提取应力、位移、损伤变量等结果。
6. 与热应力解析模型进行对比。
7. 整理论文中的模型验证和失效机理分析内容。

## 使用的 AI 模型

本项目涉及的 AI 工具包括：

- GPT 系列
- Claude 系列
- DeepSeek 系列
- Gemini 系列
- Xiaomi MiMo 系列

AI 主要用于科研辅助，而不是直接替代有限元建模和人工判断。

## 示例任务

典型任务包括：

- 生成 Abaqus 后处理脚本
- 提取路径应力数据
- 计算界面损伤指标
- 整理 Hsueh 解析热应力模型
- 对比有限元结果与解析结果
- 生成论文中的表格和分析文字

## 数据说明

由于本研究涉及未公开的课题数据，完整 Abaqus 模型文件、原始计算结果和论文正文暂不公开。本仓库仅展示简化后的研究流程、示例脚本和 AI 辅助科研的工作方式。
